# Spreads

## 1-card (clarifier)

```
[ card ]
```

Use when:
- user asks a direct yes/no question
- user wants a quick "what's the energy right now"
- follow-up to another spread ("okay one more — what should I do first?")

Read: just the card, upright/reversed, in plain language. One sentence per axis (situation / what's needed). Done.

---

## 3-card (Past / Present / Future) — DEFAULT

```
[ Past ] [ Present ] [ Future ]
```

Use when:
- question is open or exploratory
- user says "what's going to happen" or "tell me about this situation"
- any first reading

Read left → right. Each card sets up the next. Past feeds Present feeds Future.

Wiring tips:
- If all three Major → big theme, slow down, give weight
- If Future is "lighter" than Past → movement, release
- If Future is "heavier" → name it, don't sugarcoat
- Repeating suit across 3 cards → one life area dominates (call it out)

---

## 5-card (Situation)

```
[Situation]
[ Self ] [ Other ]
[Advice]
[Outcome]
```

Use when:
- user signals something big: "งาน" "ความสัมพันธ์" "ย้ายบ้าน" "ลาออก"
- the user has a specific person or choice in mind
- need nuance — Self / Other separation matters

Read:
- Situation = the outer facts
- Self + Other = both parties (or two aspects of user)
- Advice = what the cards ask of the user right now
- Outcome = the trajectory *if nothing changes* (not a verdict — a weather reading)

---

## 7-card (Horseshoe)

For complex life crossroads. Rarely needed — most sessions live well inside 3 cards.

```
[Past] [Present] [Hidden][ Obstacles | Advice | Likely outcome] [ Future ]
```

Skip unless the user is clearly at a life-shaping decision and 5 cards wouldn't carry the weight.

---

## When to break the layout

Layouts are scaffolding, not law. If the story jumps out:
- Mention it ("Card 2 and Card 4 sing the same note — aha")
- Re-interpret out of order if it reads better
- Pull a clarifying card live ("one more — *what's underneath this?*")

The cards serve the user's understanding. Not the other way around.
